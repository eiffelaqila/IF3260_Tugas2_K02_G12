Help Page

Memilih atau membuka file model objek berongga

  1. Pilih tombol Object Properties pada toolbar panel di kiri layar
  2. Pilih objek yang telah tersedia melalui dropdown shape yang ada pada Object Panel pada properties panel di kanan layar
  3. Untuk memuat file model objek berongga, pilih opsi “From file” pada dropdown shape, lalu pilih file model yang ingin digunakan melalui input file di bawah dropdown tersebut

Menyimpan hasil model

  1. Tekan tombol “Save” pada properties panel di kanan layar
  2. Simpan hasil model yang telah ditransformasi tersebut pada perangkat pengguna

Melakukan transformasi terhadap model objek

  1. Pilih tombol Object Properties pada toolbar panel di kiri layar
  2. Lakukan transformasi translasi, rotasi, atau dilatasi dengan menggunakan slider-slider yang ada pada Object Transformation Panel pada properties panel di kanan layar

Memilih proyeksi terhadap model objek

  1. Pilih tombol Camera and Projection Properties pada toolbar panel di kiri layar
  2. Pilih jenis proyeksi yang ada pada Projection Panel pada properties panel di kanan layar

Melakukan konfigurasi kamera

  1. Pilih tombol Camera and Projection Properties pada toolbar panel di kiri layar
  2. Lakukan konfigurasi kamera dengan menggunakan slider-slider yang ada pada Camera Configuration Panel pada properties panel di kanan layar

Menyalakan shading

  1. Pilih tombol Camera and Projection Properties pada toolbar panel di kiri layar
  2. Tekan tombol “Shading” untuk menyalakan atau mematikan shading terhadap model objek

Menyalakan animasi

  1. Pilih tombol Camera and Projection Properties pada toolbar panel di kiri layar
  2. Tekan tombol “Animation” untuk menyalakan atau mematikan animasi terhadap model objek

Melakukan reset view

  1. Pilih tombol Camera and Projection Properties pada toolbar panel di kiri layar
  2. Tekan tombol “Reset View” untuk me-reset view ke mode default
